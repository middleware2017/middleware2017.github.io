ACM/IFIP/USENIX Middleware 2017

Las Vegas, Nevada

Dec 11-15, 2017

Program Overview

An overview of the pre-conference program (Monday, Dec 11th and Tuesday, Dec 12th) can be foud in the table below. This includes workshops, tutorials, and the doctoral symposium. The main conference program is still being finalized, and the webpage will be updated with the final program as it becomes available.

Monday, Dec 11 (pre-conference)

EVENT SCHEDULED TIME SLOT
Tutorial - High Performance Network Middleware with Intel DPDK and OpenNetVM morning (AM)
Tutorial - Trusted Execution of Software using Intel SGX afternoon (PM)
Doctoral Symposium all day (AM-PM)
IoT Federated Workshops: M4IoT & MECC all day (AM-PM)
DIDL: Workshop on Distributed Infrastructures for Deep Learning afternoon (PM)

Tuesday, Dec 12 (pre-conference)

EVENT SCHEDULED TIME SLOT
Tutorial - SMACK stack 101: Building Fast Data stacks morning (AM)
Tutorial - Istio Service mesh for more robust, secure and easy to manage microservices afternoon (PM)
Active: International Workshop on Active Middleware on Modern Hardware all day (AM-PM)
SERIAL: ScalablE and Resilient InfrAstructures for distributed Ledgers all day (AM-PM)
WoSC: Workshop on Serverless Computing all day (AM-PM)
ARM: Adaptive and Reflective Middleware morning (AM)
